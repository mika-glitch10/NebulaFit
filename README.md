## Overview
NebulaFit is a beginner-friendly fitness application that provides **AI-assisted personalized exercise recommendations** based on user inputs.  
The app is designed as a lightweight prototype for hackathons, academic projects, and concept demonstrations.

## Problem Statement
Many beginners struggle to choose the right workouts due to lack of guidance and personalization. NebulaFit addresses this gap by using AI logic to suggest exercises tailored to individual needs.

## AI Integration
NebulaFit integrates **AI-driven decision logic** to generate personalized exercise routines.  
Based on parameters such as:
- Fitness goal  
- Body metrics (e.g., BMI category)  
- Activity preference  

The AI module analyzes inputs and recommends suitable workouts.  
The system is **API-ready**, allowing future upgrades with advanced AI models or cloud-based AI services.
