# NebulaFit

## Overview
NeublaFit is a beginner-friendly fitness app that generates personalized exercise routines using AI logic and user inputs.

## Features
- Personalized workout recommendations
- BMI-based fitness suggestions
- Simple and clean UI
- No backend required (prototype version)

## Built With
- MIT App Inventor
- AI logic (rule-based / API-ready design)
